# Lean IDS Storybook

Live documentation and component library for Lean IDS Design System.

## 🎨 What's Inside

This repository contains the built Storybook for Lean IDS, featuring:

- **30+ React Components** - Button, Input, PageLayout, TopHeader, SideNavigation, and more
- **Design Tokens** - Colors, Typography, Spacing, Borders, Shadows
- **Interactive Examples** - 50+ component stories with live code
- **Documentation** - Usage guidelines and best practices
